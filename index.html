Simple and nice

dynamic-background it's a web component which lets you have a beautiful landing page divided in full-screen sections, where the page background changes depending which section is currently shown on screen.

Ready to use

You only need to import the dynamic-background and background-element web components and use them like in the following example: <dynamic-background>
  <background-element type="background" background="URL">
  /* content */
  </background-element>
</dynamic-background>

There's no limit!

You can use as many elements as you want.

Download it

Install via bower: bower install --save dynamic-background
